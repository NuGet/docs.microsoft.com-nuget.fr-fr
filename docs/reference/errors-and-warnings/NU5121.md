# <a name="nuget-warning-nu5121"></a>AVERTISSEMENT NuGet NU5121
<pre> 'content' assets will not be available when the package is installed after the migration.</pre>

### <a name="issue"></a>Problème

Le package contient des fichiers dans un `Content` dossier. Ces ressources ne sont pas disponibles si le package est installé dans un type de référence de package de projet.


### <a name="solution"></a>Solution

Demandez à l’auteur du package de corriger le package de sorte qu’il ne contienne `Content` plus de dossier. Vous pouvez en savoir plus à ce sujet dans les [documents NuGet](https://docs.microsoft.com/en-us/nuget/consume-packages/migrate-packages-config-to-package-reference).
